# v0.1.0 版本说明

发布日期：`2026-03-15`

## 1. 版本定位

`v0.1.0` 为黄金价格走势分析系统首个可运行版本，已完成从数据采集、新闻入库、因子分析、日报生成、评分复盘到前端可视化展示的主链路闭环。

## 2. 已交付能力

- 人民币/克黄金价格实时采集、写库、K 线聚合与 SSE 推送
- 国内外黄金相关新闻抓取、去重、分类、情绪与影响因子映射
- 10 个核心黄金影响因子初始化、历史快照预热与最新快照更新
- 规则引擎版每日报告生成、结构化预测入库与次日评分
- 历史准确率曲线统计与首页展示
- 自动任务定义中心、新闻/因子/报告/评分自动调度、失败重试与告警钩子
- 前端首页金融风 Dashboard、加载态/空态/错误态与局部重试交互

## 3. 验证结果

- `go test ./...` 通过
- `npm run build` 通过
- `make check-sync` 通过
- 本地 smoke test 已验证：
  - `GET /api/v1/health`
  - 前端开发服务入口 `http://127.0.0.1:5173/`

## 4. 当前边界

- 日报生成、新闻摘要、情绪和评分仍为本地规则引擎版，尚未切到真实 AI 模型
- 部分长周期价格与宏观因子仍带有预热/回退数据特征
- 后台任务中心当前为无鉴权版本，适合开发环境和单机部署

## 5. 下一阶段建议

1. 接入真实 AI 模型替换规则引擎版日报、摘要和评分解释
2. 对接真实宏观数据源，增强因子质量与相关性
3. 增加后台鉴权、部署脚本和外部告警渠道
4. 扩展前端多页能力、筛选器和详情页交互
